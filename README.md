# AfriTalent

## Présentation

AfriTalent est une plateforme web conçue pour mettre en relation les entreprises et les talents africains. Le site permet de découvrir des freelances dans différents domaines tels que le développement web, le marketing digital et la communication.

## Objectifs

- Valoriser les talents africains.
- Faciliter le recrutement de freelances.
- Offrir une interface moderne, responsive et intuitive.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Git & GitHub

## Fonctionnalités

- Navigation responsive
- Catalogue de freelances
- Système de filtrage par catégorie
- Formulaire de contact
- Interface Dark Mode
- Animations et effets visuels
